Key Project 01

Core Behavior Improvement
(핵심 행동 및 리텐션 개선)

"게이미피케이션 기반 리워드 최적화로 마케팅 효율을 극대화했습니다."

Problem

높은 신규 유저 획득 비용(CAC) 대비 체리피커 유입으로 인해 실질적 리텐션이 저하되고 마케팅 예산 효율성이 악화되었습니다.

Hypothesis

단순 보상이 아닌 '게임(돈나무)'과 '친구 초대'를 결합한 리워드 생태계를 구축하면, 자발적 바이럴과 매일 방문하는 습관이 형성될 것입니다.

Action

  • 행동 기반 데이터 분석을 통해 사용자별 보상 단가 차등화
  • 통합 적립금 시스템 및 포인트 샵 연동

Result

  • DAU 483% 성장 및 가입자 1,532% 폭증
  • 업계 최저 수준인 CAC 1,200원 달성

Key Project 02

Live Service UX Improvement
(진입 장벽 제거 및 인증 혁신)

Problem

'가입 → 계좌개설 → 연동'의 3중 허들과 외부 인증(카카오페이) 의존도로 온보딩 과정에서 90% 이상의 이탈이 발생했습니다.

Hypothesis

외부 앱 이동을 최소화하고 자체 인증(PIN/생체)을 도입하여 단계를 축소하면, 진입 속도가 향상되고 이탈률이 50% 이상 감소할 것입니다.

Action

  • 3단계 과정을 1단계로 통합하는 '평지형' UX 설계
  • 자체 보안 키패드 및 생체 인증 로직 구현

Result

서비스 진입 속도 90% 개선 및 VOC 획기적 감소
